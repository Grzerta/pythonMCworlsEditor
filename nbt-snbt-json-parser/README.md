# nbt-snbt-json-parser

## Overview
This project provides a parser for SNBT (Stringified NBT) and JSON formats using the `nbtlib` library. It allows users to easily convert between these formats and manipulate NBT data.

## Installation
To install the required dependencies, run the following command:

```
pip install -r requirements.txt
```

## Usage
To run the SNBT/JSON parser, execute the following command:

```
python src/main.py
```

Follow the on-screen instructions to parse SNBT or JSON data.

## Testing
To run the unit tests for the NbtParser class, navigate to the `src/tests` directory and run:

```
pytest test_nbt_parser.py
```
